Beta-EstradiolDB is a database containing the RNA-Seq data for the Wang et al (2010) article, using a BowTie-Cufflinks analysis pipeline.

 

 

Summary of Analysis

This website provides a summary of analysis detailing the experimental source of the data, chosen pipeline and reason for the use of Beta-Estradiol.

 

Gene Search

The Gene Search function provides a visualisation of the analysis provided per gene. The annotation mapping link also allows visualisation of mapped associated genes and splice variants.

 

Gene List

The Gene List function provides a list of expressed genes with linked fastq result files.

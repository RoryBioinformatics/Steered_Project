FingeRNAils is a database containing the RNA-Seq data for the Wang et al (2014) article, using a TopHat-Cufflinks analysis pipeline.

 

 

Summary of Analysis

This website provides a summary of analysis detailing the experimental source of the data, chosen pipeline and reason for the use of Beta-Estradiol.

 

Gene Analysis

The analysis and visualisation section provides a link to a server which allows for the visualisation of queried genes, alongside comparison of the gene set.

 

Annotation Mapping

The annotation mapping link also allows visualisation of mapped queried genes and splice variants in the IGV browser.
